Ripple - Responsive Storage
==================================

QuickStart
==========

1. Download Ripple::

    $ git clone https://github.com/ryanchard/ripple.git

2. Install requirements::

    $ pip3 install -r requirements.txt

3. Install Ripple::

    $ python3 setup install

4. Run it::

    $ ripple
