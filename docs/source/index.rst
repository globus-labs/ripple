Ripple Documentation
==================================

Quickstart
==========

Installing
----------

1. Download Ripple::

    $ git clone https://github.com/ryanchard/ripple.git

2. Install::
    
    $ cd ripple
    $ python3 setup.py install

3. Run ripple::

	$ ripple

Requirements
============

Ripple requires the following:

* Python 3.5+

For testing:

* nose
